# TP 5 : Intégration de Spring et Hibernate 
##  Description :
#### Ce projet illustre l’intégration du framework **Spring** avec **Hibernate** afin de créer une application Java capable de gérer la persistance des données dans une base **MySQL**.L'application met en place une architecture simple basée sur le modèle **DAO (Data Access Object)** permettant d’effectuer des opérations **CRUD (Create, Read, Update, Delete)** sur une entité `Product`.
#### Spring est utilisé pour gérer :
#### - l’injection de dépendances
#### - la configuration de l’application
#### - la gestion des transactions

#### Hibernate est utilisé pour :
#### - le mapping objet-relationnel (ORM)
#### - l’interaction avec la base de données.
##  Objectifs :
#### Les principaux objectifs de ce TP sont :
#### - Comprendre la gestion des dépendances avec Maven
#### - Configurer Spring pour l’injection de dépendances
#### - Intégrer Hibernate pour la persistance des entités
#### - Implémenter une architecture DAO
#### - Connecter une application Java à une base de données MySQL
#### - Mettre en place un CRUD complet
## Technologies utilisées :
- **Java**
- **Spring Framework**
- **Hibernate ORM**
- **Maven**
- **MySQL**
- **JPA (Java Persistence API)**

##  Conclusion :
#### Ce TP permet de comprendre comment intégrer **Spring et Hibernate** pour développer une application Java capable de gérer la persistance des données.
#### Il montre également comment :
#### - configurer un projet **Maven**
#### - connecter une application à **MySQL**
